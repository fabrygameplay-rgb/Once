# Sarracenia Leucophylla - Mi Planta Favorita

Proyecto web estático desarrollado con **HTML5 y CSS3** que presenta información descriptiva y visual sobre la especie carnívora Sarracenia leucophylla.

---

## Descripción

Sitio informativo dedicado a la planta carnívora **Sarracenia leucophylla**, conocida como “planta de jarro blanco”.

Incluye:

- Información general sobre la especie
- Características distintivas frente a otras Sarracenias
- Lista descriptiva de rasgos biológicos
- Galería visual organizada en sistema grid
- Sección de referencias externas

---

## Objetivo del Proyecto

Practicar:

- Estructura semántica en HTML
- Maquetación con CSS Grid
- Diseño responsive con múltiples breakpoints
- Adaptación tipográfica según tamaño de pantalla
- Organización visual limpia sin frameworks

---

## Tecnologías Utilizadas

- HTML5 semántico
- CSS3
  - Grid Layout
  - Media Queries
  - Tipografía adaptable
  - Ajustes progresivos por resolución
